Actor Network Analysis

Connections in Hollywood

Posted by Christian Payne on August, 2023

Who are the most connected actors in Hollywood? With network analytics, we can identify the most connected actors and explore other interesting characteristics of the wider acting network.

Co-star Networks

Given the huge number of films, we'll be using the list of the of all time from IMDB. With this list, we'll scrape the cast lists and construct a network by creating connections between co-stars.

For example, with the cast lists for Star Wars and Indiana Jones we can construct a network of co-stars. Looking at the resulting network below, our eyes are initially drawn to the two large clusters and the smaller cluster. These large clusters contain the casts for the two films whilst the smaller one contains the cast who have appeared in both. This of course includes Harrison Ford as well as a few stuntmen.

Now we have this example we can build the network and start investigating it's structure.

Building the Network

The resulting network is huge. Specifically, the scraped data contains 178,214 nodes (actors) and 31,050,918 edges (co-star relationships). The size and density of the network means that visualising it poses a challenge. However, what we can do is break the network down into sections and visualise those sections.
